# Dirt!
A dirty boilerplate for prototyping an HTML5 Site with SASS styles and plain ol' es5 JS

### Flow
``git clone https://github.com/pstrawberriedev/dirt.git``  
``cd dirt && npm install``  
``npm start`` -or- ``gulp``

### Included
- An HTML file
- A SASS file with an import
- A JS file with a console.log

### Not Included
- Everything else
